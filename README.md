# アーサー学院 Webサイト（AG_website）

長野県内に5校舎を展開する個別指導塾「アーサー学院」のWebサイト。
**Web経由のお問い合わせを増やすこと**を最重要目的（KPI）とした、純HTML/CSS/JS の静的サイトです。

- 現ステージ：**7/3提案用のデザイン案サンプル**（相見積もり中）。本番運用前の段階です。
- フレームワーク不使用。ビルド工程なし。GitHub Pages で公開する想定。
- 制作フォーマットは個人サイト `my-website` の `SITE_TEMPLATE.md`（白ベース＋単色アクセント、フラット、ホバー時のアクセント線モチーフ、§7禁止事項）に準拠。
- デザインの方向性は競合 `izumijuku.co.jp` を**参考（複製ではない）**に、配色は既存サイト `arthurgakuin.com` から抽出した**ブランドブルー（#578fbd 系）**をオリジナル実装で再構成。

## 公開URL（要確認）

`https://shunta-tf.github.io/AG_website/` を**仮の公開URL**として canonical / OGP / sitemap / robots に設定しています。
実際の公開先（github.io のサブパス、または独自ドメイン）が決まったら、全ページの `canonical` / `og:url` / `twitter` と `sitemap.xml` / `robots.txt` / JSON-LD の URL を一括で差し替えてください。

## 技術メモ

- **デザイントークン** は `css/style.css` の `:root` に集約。色は `--accent`（#2c6fa6）/ `--accent-light`（#578fbd 抽出色）/ `--accent-dark` / `--accent-deep`（ヒーロー）。
- **タイポ**：Noto Sans JP（日本語）＋ Montserrat（英字ラベル）。Google Fonts を `<head>` で読み込み。
- **アクセシビリティ**：skip-link、`<main id="main">`、`aria-expanded`/`aria-current`、`prefers-reduced-motion` 対応、画像 `loading="lazy"`、地図 iframe に `title`。
- **レスポンシブ**：1024 / 768 / 480px。768px 以下でナビがハンバーガーに切り替わり、ヘッダーのCTA・電話はモバイルメニュー内に格納。
- **お知らせのタグ**：各 `.news-item` に `data-tags="〔スコープ〕 〔カテゴリ〕"`。スコープ＝`zenkou`(全校)/`kousha`(校舎別)、カテゴリ＝`info`/`event`/`closed`/`result`。`news.html` の絞り込みボタンと連動。本部一括発信の見せ方として実装。

## フォーム（Formspree）設定方法

`contact.html` の `<form action="https://formspree.io/f/〔FORM_ID〕">` を実際の Formspree エンドポイントに差し替えてください。
- Formspree でフォームを作成 → 発行された `https://formspree.io/f/xxxxxxxx` を `action` に設定。
- 送信時のメール通知先（クライアントのアドレス）を Formspree 側で設定。
- 現状は `〔FORM_ID〕` のままなので、`js/main.js` がプレースホルダを検知して**送信成功画面のデモ表示**を行います（実送信はされません）。エンドポイント設定後は AJAX で実送信されます。
